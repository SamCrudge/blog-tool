# blog-tool

Simple blog tool that manages posts.
PR'd by peers.

Once cloned run
``` Composer install ```

then run
```Composer start```

<br>
<br>
<br>

To create a new posts in DB use:
```/create```

To read all posts in DB use:
```/read```

To update an existing posts in DB use:
```/update```

To soft delete a posts in DB use:
```/delete```
